# Benchmark Results

This repository contains benchmark results for [ClassiFaaS](https://github.com/persists/ClassiFaaS). ClassiFaaS tries to identify different hardware on various serverless platforms.

## Unified Time Results

The `unified-time-results` directory contains the results of a benchmark which is executed at various cloud providers and regions. The Benchmark runs where timed to be executed at the local time of the respective region. The results are organized by cloud provider and region.
